## What's in the box

A single public function:

```python
from pacs008_loader_mt103 import parse_mt103

records = parse_mt103(mt103_text)   # -> one-element list[dict]
```

Each record's keys are exactly the flat-record fields the
[`pacs008`](https://github.com/sebastienrousseau/pacs008) library
validates against the `pacs.008.001.08` JSON schema, so the output
feeds straight into pacs.008 generation.

## Why this exists

SWIFT MT-MX coexistence for cross-border payments ends in **November
2025**. This loader bridges the window where upstream systems still
emit MT103 but downstream tooling expects ISO 20022 pacs.008.
